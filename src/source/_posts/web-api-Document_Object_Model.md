# DOM

> MDN: https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model

**DOM**用来描述文档树，与JavaScript连接起来。它是前端编程中使用最频繁的接口合集，涵盖了文档属性、节点管理、事件，难以一一列举。
